# 韩信点兵

# Description

今有物不知其数，三三数之剩二，五五数之剩三，七七数之剩二，问物几何？”按照今天的话来说：一个数除以3余2，除以5余3，除以7余4，求这个数。这样的问题，也有人称为“韩信点兵”。(来源：百度百科)

# Input

三个正整数，分别为被3、5、7的余数，以一个空格分隔。

# Output

满足条件的最小的正整数

# Sample input

```
2 3 2

```

# Sample output

```
23

```
