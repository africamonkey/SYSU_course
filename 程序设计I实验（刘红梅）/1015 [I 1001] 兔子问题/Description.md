# [I 1001] 兔子问题

# Description
TA家里养了一对兔子，从出生后第3个月起每个月都生一对兔子，小兔子长到第三个月时就会变成成年兔子，从这个月起这对兔子每个月又生一对兔子，假如兔子都不死，问第n个月TA家里有多少只兔子？
# Input
本体包含多组测试数据。以EOF或者-1结束。
每组测试数据包含一个正整数n，表示需要计算的月数。
# Output
每组测试数据的结果占一行。
每行输出当月TA家中的兔子数量。
# Sample Input 1
```
2
3
4
-1

```

# Sample Output 1
```
2
4
6

```

# Sample Input 2
```
2
3
4

```

# Sample Output 2
```
2
4
6

```
